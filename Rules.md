## Rules

For a better experience, follow these simple rules:

1. **Don't log in as root**  
- Pressing `Ctrl + C` on the login screen will automatically log you in as root.
- Exiting the user you logged in as from that login interface will drop you to root too.

2. **Use only the provided tools**  
- You can use external tools if you really want to, but everything you need is already installed in the system.

3. **Be respectful of ECDH infrastructure**  
- Some challenges are connected to ECDH properties, like the official website.  
- Do **not** use destructive tools—this CTF was designed to be solved without them.
  
4. **Switch users using `su username`**
- Instead of exiting the docker image and using the loging interface again, simply use the `su` command.

**Tip:** Save every flag you find, could be useful.
